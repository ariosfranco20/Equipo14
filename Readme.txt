La aplicación va a estar alojada en un servidor web el cual tendrá dos componentes principales con sus respectivos artefactos. El primer módulo o componente es el de autenticación, el cual tiene una relación de dependencia con los artefactos de interfaz de registro, interfaz de Login y Base de Datos. El segundo modulo es el de gestión de paquetes, el cual tiene una relación de dependencia con los artefactos de interfaz de Crear Orden, interfaz de actualizar orden y Base de Datos. Se podrá acceder a la aplicación desde cualquier dispositivo con navegador web y conexión a internet.
